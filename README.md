<h1>Web-приложение To-Do List </h1>
Реализован CRUD функционал: списки задач с возможностью добавить к ней категорию, возможностью фильтровать.
Реализована аутентификация пользователей и пагинация. Интерфейс сделан с помощью Laravel blade. <br> <br>
Также есть API, по которому можно выполнить все те же операции(так же требующие аутентификации) что и в web.<br>

<br>Установка проекта:<br>1. Настроить подключение к БД(.env)<br>2. composer i<br>3. npm i<br>4. php artisan migrate<br>5. php artisan serve
